Instructions for Truth Table

1. 1st choose the correct combinational logic above

2. columns are from A-D and rows are 1-8, for example if you want to chooose a certain output, say the 1st logic gate is B1

3. 1st Choose the inputs and logic gate on row 1, it you require extra Gates , you may utilise other rows,

4. Select the number of Input your combinational Logic Circuit has, (A,B,C,D). FOR NOW THE ALGORITHM CAN ONLY TAKE 4 INPUTS

5. Then click generate!

6. If a column title is red, that means you have not selected the input variable

8. ignore empty cells in the Truth-table

9. R3o represents the output of row 3, same thing as for others

10. if you want auto generated inputs, click "auto generate"

Note: all input fields will be cleared

11. if a certain variable is no filled in, it will be auto generated.

12. !! A is the master input, if you are not auto generating, then it must be filled in.
